# Horizontable

> Horizontable is a React component to scroll with mouse wheel, drag, and touch events horizontally

## How use

```jsx
<div className="parent">
  <Horitonzable>
    <div className="children">
      <div>Scroll me 😍</div>
      <div>Drag me 😎</div>
      <div>Scroll me 🎉</div>
    </div>
  </Horitonzable>
</div>
```

[See demo](https://horizontable.surge.sh)
